Detailed Description

Implements a C# promise.

Implements a non-generic C# promise, this is a promise that simply resolves without delivering a value.

https://developer.mozilla.org/en/docs/Web/JavaScript/Reference/Global_Objects/Promise

Add a callback that chains a non-value promise.

ContinueWith callbacks will always be called, even if any preceding promise is rejected, or encounters an error. The state of the returning promise will be based on the new non-value promise, not the preceding (rejected or resolved) promise.
