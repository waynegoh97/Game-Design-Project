bool FullSerializer.fsGlobalConfig.AllowInternalExceptions = true
static

If exceptions are allowed internally, then additional date formats can be deserialized.

Note that the Full Serializer public API will not throw exceptions with this enabled; errors will still be returned in a fsResult instance.
