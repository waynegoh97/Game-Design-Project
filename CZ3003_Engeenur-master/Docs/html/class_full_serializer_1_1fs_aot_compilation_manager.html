Type  type

static

Ahead of time compilations that are available.

The type maps to the object type the generated converter will serialize/deserialize, and the string is the text content for a converter that will do the serialization.

The generated serializer is completely independent and you don't need to do anything. Simply add the file to your project and it'll get used instead of the reflection based one.
