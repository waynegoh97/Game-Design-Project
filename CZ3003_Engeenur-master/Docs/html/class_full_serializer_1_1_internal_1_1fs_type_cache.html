Detailed Description

Caches type name to type lookups.

Type lookups occur in all loaded assemblies.

Find a type with the given name.

An exception is thrown if no type with the given name can be found. This method searches all currently loaded assemblies for the given type. If the type cannot be found, then null will be returned.
